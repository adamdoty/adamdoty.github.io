Headings

Tag Name Description
<h1> heading1 The first and largest heading. Only use 1 time, preferably in <header>
<h2> heading2 The second heading.
<h3> heading3 The third heading.
<h4> heading4 The fourth heading.
<h5> heading5 The fifth heading.
<h6> heading6 The sixth heading.
Tips
To create a custom heading, use the <span> element and style it with a css class selector.
Use the semantic <header> element for titles and other headings that go at the top of a page.

Semantic HTML

Tag Name Description
<header> header Contains the heading title tag and other heading tags at the start of a web page.
<main> main Used within html <body> element. Indicates the main content of the webpage.
<nav> navigation Holds links to other web pages. Often used within the semantic <header> element.
<section> section Similar to <div>, used for indicating a group of content.
<footer> footer Contains links, address, copyright, contact information and other details at the bottom of a page.
Tips
Semantic html elements allow other developers to understand the author's intention easier than just using css class selectors.
These elements also assist with accessibility tools like screen readers.
